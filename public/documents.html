Anime Quotes API Documentation

Base URL

The base URL for accessing the API endpoints is https://anime-blossom-quotes.onrender.com.

Endpoints

Get Random Quote

  • URL: /api/quotes/random
  • Method: GET
  • Description: Retrieves a random quote from the collection of anime quotes.
  • Response: A JSON object containing a random quote.

Get All Quotes

  • URL: /api/quotes
  • Method: GET
  • Description: Retrieves all available quotes from the collection.
  • Response: An array of JSON objects, each representing a quote.

Get Quote by ID

  • URL: /api/quotes/:id
  • Method: GET
  • Description: Retrieves a quote by its ID.
  • Parameters:
    • id (integer) - The ID of the quote to be retrieved.
  • Response: A JSON object containing the requested quote if found; otherwise, an error message indicating an invalid ID.

Get Quote by Character Name

  • URL: /api/quotes/character
  • Method: GET
  • Description: Retrieves a quote based on the character name.
  • Parameters:
    • name (string) - The name of the character to search for.
  • Response: A JSON object containing the quote related to the specified character if found; otherwise, an error message indicating an invalid character name.

Error Handling

  • If an invalid ID is provided while querying by ID, the response will contain an error message specifying an invalid ID.
  • If an invalid character name is provided while querying by character, the response will contain an error message specifying an invalid character name.
